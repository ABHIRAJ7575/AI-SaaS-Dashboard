## Features (as per GEMINI.md)

### Design System & Visual Identity
- Color Palette: Deep Space Black, Dark Slate, Neon Violet, Electric Blue, etc.
- Typography: Poppins, Inter, JetBrains Mono
- Glassmorphism Effects, Neon Accents, Motion Design, Custom Cursor

### Page Specifications
- **index.html:** Landing/Onboarding page with Hero, Features, Stats, Testimonials, CTA.
- **login.html & register.html:** Authentication pages with split-screen layout, floating labels, real-time validation, social login.
- **dashboard.html:** Main Dashboard with Top Nav, Sidebar, Welcome Section, Stats Cards, Interactive Charts, Recent Activity, Draggable Widgets, FAB.
- **users.html:** User Management with Search, Filters, Data Table (sortable, searchable, paginated), Modals.
- **settings.html:** Settings & Preferences with Tabbed Layout (Profile, Account, Appearance, Notifications, Privacy, API).
- **help.html:** Help Center with Search, Category Cards, FAQ Accordion, Contact Form.

### Animation Specifications
- Page Transitions, Element Reveal (Scroll-based), Hover Effects, Loading States, Toast Notifications.

### Reusable Component Library
- Buttons, Input Fields, Cards, Modals, Tooltips, Badges, Progress Bars.

### Functionality & Features
- Authentication, Theme System, Global Search (Command Palette), Notifications System, Data Persistence (localStorage), Mock Data.

### Responsive Design
- Mobile-first approach with breakpoints for small tablets, tablets, small laptops, desktops, large desktops, and ultra-wide displays.

### Performance Optimization
- CSS, JavaScript, Loading Strategy, Animation Performance optimizations.

### Code Quality Standards
- HTML5 Semantics, ARIA labels, BEM naming, CSS variables, ES6+ JavaScript, JSDoc comments, Error handling.

## External Resources (CDNs)
- Chart.js

- Google Fonts (Inter, Poppins)

## Critical Reminders
- No placeholders, real data, clean code, performance (60fps animations), consistency, attention to detail, mobile first, accessibility, error handling, testing.
